# With You. - 臨床的根拠に基づくスクリーニングシステム

## 🎯 プロジェクト概要

学校向けメンタルヘルスアプリ「With You.」の管理者側UIです。
生徒の気分・睡眠・相談データから、**臨床的根拠に基づいて**支援が必要な生徒を自動スクリーニングします。

---

## 🔬 エビデンスベース設計

このシステムは以下の研究・理論に基づいています：

### 1. 国立成育医療研究センター研究（2025）
**身体症状数と抑うつリスクの相関**
- 頭痛・腹痛・背部痛・めまいの症状数で抑うつリスクを判定
- 症状3つで**11.3倍**、4つで**16.4倍**のリスク上昇

### 2. 行動活性化療法
- 過眠や回避行動パターンの検出
- 活動性低下のサイクル早期発見

### 3. 思春期うつ病研究
- 持続的な低気分（3日連続など）の重要性
- 睡眠障害の早期指標としての価値

---

## 📊 スクリーニングレベル

### 🔴 レベル1: 緊急対応が必要
- 自殺リスク・危険ワード検出
- 重度の気分低下 + 睡眠障害
- 身体症状3-4個 + 抑うつ

**推奨**: 即日中の個別面談・保護者連絡

### 🟡 レベル2: 注意が必要
- 3日連続の気分😟
- 慢性的な睡眠不足
- 身体症状2個
- 過眠（回避行動の可能性）

**推奨**: 週1回の経過観察・声かけ

### 🔵 レベル3: 様子見
- 単発の気分😟
- やや短い睡眠（5-6時間）
- 身体症状1個

**推奨**: 月1回の全体モニタリング

---

## 🚀 実装済み機能

### ✅ データ取得
- Firestoreから全履歴データ取得
- ユーザーごとの履歴整理
- 相談メッセージの統合

### ✅ スクリーニング
- 多段階判定ロジック
- 危険ワード検出
- 身体症状カウント
- 連続パターン分析
- 過眠・回避行動検出

### ✅ 可視化
- サマリーカード（レベル別集計）
- 低気分率ヒートマップ
- クラス別サマリーテーブル
- エビデンス表示セクション
